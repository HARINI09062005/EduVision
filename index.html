How EduVision Works

Keyboard Shortcuts

  • Alt + H - Home
  • Alt + M - Main menu
  • Alt + A - Accessibility options
  • Alt + C - Go to Create page

Voice Commands

  • "Read page" - Screen reader activation
  • "Navigate to [section]" - Jump to section
  • "Increase contrast" - Toggle high contrast
  • "Go to create" - Navigate to Create page
